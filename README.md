# Ariel- Crowdsec

## How do I install these formulae?

`brew install ariel-/crowdsec/<formula>`

Or `brew tap ariel-/crowdsec` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ariel-/crowdsec"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
